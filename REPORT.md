## 1. Project Overview

This project demonstrates the creation of a personal portfolio website using GitHub Pages and the Jekyll static site generator. The website consists of 5 pages showcasing skills, projects, and contact information.

## 3. Website Structure

The website contains the following pages:

1. **Home (index.md)** - Welcome page with navigation to other sections
2. **About (about.md)** - Background information and education details
3. **Projects (projects.md)** - Showcase of completed and ongoing projects
4. **Skills (skills.md)** - Technical skills and competencies
5. **Contact (contact.md)** - Contact information and LinkedIn profile

---

## 4. Technologies Used

- **Jekyll** - Static site generator
- **GitHub Pages** - Hosting platform
- **Markdown** - Content formatting
- **YAML** - Configuration (_config.yml)
- **Git** - Version control
